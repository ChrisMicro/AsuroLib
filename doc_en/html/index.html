Introdution

The Asuro Library is a collection of C-functions for the Asuro robot. The Asuro have been developed at DLR (German Institute for für Aerospace) http://www.dlr.de at Oberpfaffenhofen/Bavaria.

The library ist based on the original DLR library from Jan Grewe/Robin Gruber. Users of the german Roboternetz forum http://www.roboternetz.de/phpBB2 have expanded the library.

Installation

There are 3 choices howto downzuload and install the Asuro Library.

  • Download the AsuroLib-vXXX-Setup.exe Installer File. Just double-click the file AsuroLib-vX.XX-Setup.exe to strat the installation.
  • Download the AsuroLib-vXXX.zip Zip file. Extract the zip-Files to a directory of your choice
  • Download via Subversion (SVN). You will need a SVN Client p.e. SmartSVN http://www.syntevo.com/smartsvn/ or TortoiseSVN http://tortoisesvn.net/
  SVN Einstellungen:
  Hostname: asuro.svn.sourceforge.net
  Port: 443
  Protocol: HTTPS
  Repository Path: /svnroot/asuro
  Username: SourceForge.net username for SVN write access, none for read access
  Password: SourceForge.net user password for SVN write access, none for read access

Directory strukture

After installation you will a find the following directory structure. 

  C:\ASURO_SRC\AsuroLib                     the installation directory
                       \doc_en              the document diretory for the english HTML and HTML Help files
                       \lib                 Library sourcecode files and object library
                       \lib\inc             Library header files
                       \examples            Example projects
                       \examples\FirstTry   Example project FirstTry
                       \examples\SelfTest   Example project SelfTest
